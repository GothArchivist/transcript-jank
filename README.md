# Transcript Jank
Scripts and templates for working with transcripts. The following scripts and related instructions are in the following folders:

**Compare The Transcripts**
Some super basic, janky scripts for comparing transcripts generated by different systems.
**Reorder .docx**
Converting and brute forcing incorrectly ordered transcripts into the right format in a particularly convulated way.
**Sensitive Language Alerting**
Materials for testing UTF-8 encoded documents for sensitive language, e.g. ethnic slurs.
